# claude-code与figma的使用

claude plugin install figma@claude-plugins-official

## claude-code 代码到 figma原型
从 Claude Code 捕获页面、状态和流程——可在 Figma 中立即编辑

- 要打开前端 dev，有页面
- 提示词： 将 xxx 页面 发送到 figma


## figma设计 到claude-code
复制figma中的设置文件url, 在claude中写入类似这样的提示词：

`使用此 Figma 文件：<Figma 文件 URL>，创建一个新页面，并使用我们现有的组件构建一个具有自动布局的设置屏幕。`
